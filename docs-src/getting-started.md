# Getting Started

Welcome to Flint! This guide will help you get started with Flint and learn the basics of note-taking with AI assistance.

## Installation

Download the latest release for your platform:

- **Windows**: Download the `.exe` installer
- **macOS**: Download the `.dmg` file
- **Linux**: Download the `.AppImage` or `.deb` package

See the [Installation Guide](/guides/installation) for detailed instructions.

## First Steps

### Creating Your First Note

1. Launch Flint
2. Click the "New Note" button or use `Cmd+N` (Mac) / `Ctrl+N` (Windows/Linux)
3. Start writing in the editor
4. Your note is automatically saved

### Basic Navigation

- **Daily Notes**: Access today's note from the sidebar
- **Search**: Use `Cmd+P` / `Ctrl+P` to quickly search and open notes
- **AI Agent**: Click the AI button or use `Cmd+K` / `Ctrl+K` to open the AI assistant

## Key Concepts

### Notes

Notes are the basic building block in Flint. Each note can contain:

- Rich text content using markdown
- Tags for organization
- Links to other notes
- Custom metadata

### Tags

Organize your notes with tags:

- Add tags using `#tagname` syntax
- Browse notes by tag in the sidebar
- Combine multiple tags to filter notes

### AI Agent

The AI agent can help you:

- Write and edit content
- Summarize information
- Answer questions about your notes
- Generate ideas and outlines

See the [AI Agent](/features/agent) documentation for more details.

### Review System

Flint includes a spaced repetition system to help you remember important information:

- Mark notes or sections for review
- Schedule reviews based on spaced repetition algorithms
- Track your learning progress

See the [Review System](/features/review-system) documentation for more details.
