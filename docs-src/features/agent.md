# AI Agent

Flint includes an integrated AI agent that can help you with writing, research, and information processing.

## Overview

The AI agent is a conversational assistant that:

- Understands the context of your notes
- Can help you write and edit content
- Answers questions and provides information
- Assists with brainstorming and ideation

## Opening the Agent

Access the AI agent by:

- Clicking the AI button in the toolbar
- Using the keyboard shortcut `Cmd+K` / `Ctrl+K`
- Selecting text and choosing "Ask AI" from the context menu

## Features

### Contextual Assistance

The AI agent has access to:

- The current note you're editing
- Selected text for targeted help
- Your conversation history

### Common Tasks

#### Writing Assistance

- Draft new content
- Expand on ideas
- Rewrite or improve existing text
- Fix grammar and style

#### Research & Learning

- Answer questions on any topic
- Explain complex concepts
- Summarize information
- Generate study questions

#### Creative Tasks

- Brainstorm ideas
- Generate outlines
- Create examples
- Suggest alternatives

## Configuration

### AI Provider

Flint supports multiple AI providers:

- OpenAI
- Anthropic Claude
- OpenRouter (access to multiple models)
- Local models

See the [Configuration Guide](/guides/configuration) for setup instructions.

### Model Selection

Choose the model that best fits your needs:

- Faster models for quick responses
- More capable models for complex tasks
- Cost considerations
