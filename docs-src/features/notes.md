# Notes & Organization

Flint provides a powerful yet simple system for creating and organizing your notes.

## Creating Notes

There are several ways to create a new note:

- Click the "New Note" button in the sidebar
- Use the keyboard shortcut `Cmd+N` (Mac) / `Ctrl+N` (Windows/Linux)
- Use the command palette (`Cmd+P` / `Ctrl+P`) and search for "New Note"

## Writing in Notes

### Markdown Support

Flint supports standard markdown syntax:

- **Bold** text with `**bold**`
- _Italic_ text with `*italic*`
- Lists with `-` or `1.`
- Headings with `#` symbols
- Code blocks with triple backticks

### Note Linking

Link to other notes using:

- `[[Note Title]]` syntax
- The notes will be automatically connected
- Backlinks are tracked and displayed

### Tags

Organize notes with tags:

- Use `#tagname` anywhere in your note
- Tags appear in the sidebar for quick filtering
- Multiple tags can be applied to any note

## Organization

### Sidebar Navigation

The sidebar provides quick access to:

- Daily notes
- Recent notes
- Tagged notes
- All notes list

### Search

Find notes quickly with:

- Full-text search across all notes
- Search by title, content, or tags
- Keyboard shortcut: `Cmd+P` / `Ctrl+P`

### Daily Notes

Special support for daily journaling:

- One note per day, automatically created
- Quick access button in the sidebar
- Configurable template for daily notes
