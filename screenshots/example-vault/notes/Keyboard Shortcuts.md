# Keyboard Shortcuts

Master these shortcuts to supercharge your Flint workflow.

## Navigation

| Shortcut      | Action                         |
| ------------- | ------------------------------ |
| `Cmd+K`       | Quick search / Command palette |
| `Cmd+B`       | Toggle sidebar                 |
| `Cmd+\`       | Toggle right panel             |
| `Cmd+Shift+D` | Go to Daily Notes              |

## Note Management

| Shortcut      | Action                            |
| ------------- | --------------------------------- |
| `Cmd+Shift+N` | Create new note                   |
| `Cmd+S`       | Save note (auto-saves by default) |
| `Cmd+Shift+F` | Search all notes                  |
| `Cmd+Enter`   | Follow wikilink under cursor      |

## Editor

| Shortcut      | Action            |
| ------------- | ----------------- |
| `Cmd+B`       | Bold text         |
| `Cmd+I`       | Italic text       |
| `Cmd+Shift+K` | Insert link       |
| `Cmd+[`       | Outdent list item |
| `Cmd+]`       | Indent list item  |

## Tips

- Hold `Cmd` and click a wikilink to open in a new pane
- Use `Cmd+P` to access the command palette for all actions
- Customize shortcuts in Settings if needed
