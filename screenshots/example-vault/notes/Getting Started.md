# Getting Started with Flint

This guide will help you get up and running with Flint in just a few minutes.

## Creating Your First Note

1. Press `Cmd+Shift+N` to create a new note
2. Give it a meaningful title
3. Start writing in markdown

## Connecting Ideas with Wikilinks

The real power of Flint comes from connecting your notes. Simply type `[[` followed by a note name to create a link:

- Link to existing notes: [[Welcome to Flint]]
- Create new notes on the fly: [[My New Idea]]

## Organizing with Note Types

Flint lets you organize notes into types like Projects, Meetings, and Research. Each type can have its own properties and templates.

## Using Daily Notes

The Daily view helps you capture thoughts chronologically. Each day gets its own note automatically, perfect for:

- Morning reflections
- Meeting notes
- End-of-day summaries
