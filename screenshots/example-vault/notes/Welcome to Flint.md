# Welcome to Flint

Flint is a powerful note-taking app that helps you capture ideas and connect them together.

## Key Features

- **Markdown Editor** - Write with familiar syntax and see it rendered beautifully
- **Wikilinks** - Connect notes with [[Getting Started]] to build your knowledge graph
- **Daily Notes** - Track your thoughts day by day with automatic date-based organization
- **AI Assistant** - Get help organizing, summarizing, and expanding your thoughts

## Quick Tips

1. Create your first note with `Cmd+Shift+N`
2. Use `[[wikilinks]]` to connect related ideas
3. Try the daily notes view for journaling
4. Use the search (`Cmd+K`) to quickly find any note
