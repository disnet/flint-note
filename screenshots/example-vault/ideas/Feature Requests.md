# Feature Requests

Collection of feature ideas from users and the team.

## High Priority

### Mobile App

_Requested by: Many users_

Native iOS and Android apps for note-taking on the go.

- Offline support essential
- Quick capture widget
- Sync with desktop

**Status**: In development - see [[Mobile App Launch]]

### Templates

_Requested by: Power users_

Pre-defined note structures for common use cases:

- Meeting notes
- Project briefs
- Daily journals
- Book notes

**Status**: Planned for Q2

## Medium Priority

### Calendar Integration

Connect notes to calendar events automatically.

### PDF Annotation

Highlight and annotate PDFs, link to notes.

### Graph View

Visual representation of note connections.
